# VimeoVideos

Simple library for uploading videos through Vimeo V2 API, uses OAuth 1.

## Installation

Add this line to your application's Gemfile:

    gem 'vimeo_videos'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vimeo_videos
